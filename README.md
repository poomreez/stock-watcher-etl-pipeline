# stock-watcher-etl-pipeline
An automated ETL pipeline built with Python and SQLite to fetch, transform, and store real-time financial market data for analysis.

Project Overview
This project demonstrates a foundational Data Engineering workflow by building an automated ETL (Extract, Transform, Load) pipeline. It replaces traditional file-based storage with a structured relational database to ensure data integrity and scalability.

Key Features
Data Extraction: Automated fetching of real-time stock price data via REST APIs using Python.
